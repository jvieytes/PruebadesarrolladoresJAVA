Antes de ejecutar las siguientes lineas de codigo tiene que implementar el script en mysql, luego
de ingresar el script em mysql implemente las siguiente lineas de codigo.

Build Spring Boot Project with Maven

mvn package (cmd en la carpeta del programa donde estan todos los archivos)

Run spring boot app using maven

mvn spring-boot:run (cmd en la carpeta del programa donde estan todos los archivos)

OR

java -jar target/com.azu.lista-0.0.1-SNAPSHOT.jar (cmd en la carpeta del programa donde estan todos los archivos)
